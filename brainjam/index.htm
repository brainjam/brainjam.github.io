Brain Jam

Brain Jam is a Windows solitaire game/card puzzle. The deck is initially dealt face up, so there is no luck involved once play begins. All hands are winnable. Featured in Games magazine. Based on "Baker's Game" described by Martin Gardner in Scientific American in 1968.

You can now play Brainjam in your browser at brainjam-solitaire.appspot.com.

If you want to play the orginal 16-bit Windows version, download the file brainj92.zip and install it using WinZip. If you don't have WinZip, decompress the file into a temporary directory. From there, run SETUP.EXE to install. This version will not run on a 64 bit computer.
